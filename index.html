Dijkstra's Algorithm — Execution Details

  • Initialise: Set distance to all nodes = ∞, except source = 0. Add source to priority queue.
  • Pick minimum: Dequeue the node with the smallest tentative distance (greedy choice).
  • Relax neighbours: For each unvisited neighbour, if current distance + edge weight < recorded distance, update it.
  • Repeat steps 2–3 until destination is dequeued or queue is empty.
  • Backtrack: Follow the "previous node" pointers from destination → source to get the path.
